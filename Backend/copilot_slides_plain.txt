SLIDE 2: Problem Statement & Motivation

PROBLEM:
TB & Pneumonia: Major global health issues
Limited Access: Shortage of radiologists, especially in rural areas
Time Consuming: Manual diagnosis requires expert review and hours/days of waiting
Error-Prone: Manual analysis vulnerable to human error
Need: AI-powered, user-friendly mobile solution

MOTIVATION:
Instant Result: Get immediate predictions without waiting for appointments
Accessibility: Empower patients with low-cost, user-friendly health monitoring tools
Impact: Particularly valuable in rural areas and emergency situations
Health Innovation: Contribute to improving healthcare outcomes through digital transformation
Continuous Monitoring: Track chest health improvements/deterioration over time

KEY INSIGHT:
Users can perform self-screening with chest X-rays and get instant predictions, then make informed decisions about visiting a doctor — saving critical time, especially in emergencies.

SLIDE 6: Comments Addressed - Pneumonia Classification Model

MODEL ARCHITECTURE:
Type: Custom Convolutional Neural Network (CNN)
Input Size: 180 x 180 x 3 (RGB images)
Architecture Highlights:
  - Multiple SeparableConv2D layers for efficient feature extraction
  - Batch normalization for training stability
  - Dropout layers (multiple stages) for regularization
  - Progressive downsampling through max pooling
  - Dense layers for classification

MODEL SPECIFICATIONS:
Total Parameters: 3,496,801 (13.34 MB)
Trainable Parameters: 3,494,433 (13.33 MB)
Framework: TensorFlow/Keras

PERFORMANCE METRICS:
Validation Accuracy: ~95%
Precision: High (minimizes false positives)
Recall: Very High (detects almost all pneumonia cases)
Trade-off: Recall > Precision (prioritizes sensitivity for medical accuracy)
Test Performance: ~90% accuracy on independent test set

KEY STRENGTHS:
- Lightweight model (13.34 MB) - suitable for mobile deployment
- High recall ensures pneumonia cases are not missed
- Efficient architecture with separable convolutions
- Regularization techniques prevent overfitting

TECHNICAL DETAILS:
Dataset: Chest X-Ray Images 2017 (TFRecords)
Data Split: 70% Training, 15% Validation, 15% Testing
Metrics Used: Binary Accuracy, Precision, Recall, Loss
Activation: Sigmoid (binary classification)

SLIDE 7: Comments Addressed - TB Classification Model

MODEL ARCHITECTURE:
Type: DenseNet121 Transfer Learning with Custom Head
Input Size: 320 x 320 x 3 (RGB images)
Base Architecture: Pre-trained DenseNet121 (ImageNet weights)
Architecture Highlights:
  - Dense block connections for feature reuse
  - Batch normalization and ReLU activations
  - Progressive feature concatenation
  - Custom classification head for TB detection

MODEL SPECIFICATIONS:
Base Model: DenseNet121 (pre-trained)
Custom Head: Dense layers with batch normalization
Output Activation: Sigmoid (binary classification)
Loss Function: Weighted binary cross-entropy (handles class imbalance)
Optimization: Adam optimizer with learning rate scheduling

PERFORMANCE METRICS:
Training Accuracy: ~92.54% - 92.76%
Validation Accuracy: ~78-79%
Test Accuracy: 83%
Precision & Recall: Balanced classification metrics
AUC (Area Under Curve): High discrimination between TB/Normal

CLASSIFICATION REPORT (TEST SET):
Class          Precision    Recall    F1-Score    Support
Normal         [High]       [High]    [High]      ~420 samples
Tuberculosis   [High]       [High]    [High]      ~420 samples
Accuracy:      83%

KEY STRENGTHS:
- Transfer learning from ImageNet improves generalization
- Weighted loss handles class imbalance in medical data
- Grad-CAM visualization for model interpretability
- Robust performance on independent test set
- Higher input resolution (320x320) captures fine details

TECHNICAL DETAILS:
Dataset: TB X-Ray Images (Collected & Preprocessed)
Data Split: 60% Training, 20% Validation, 20% Testing
Training Epochs: Multiple with early stopping
Data Augmentation: Random rotations, shifts, zooms
Metrics: Accuracy, AUC, Weighted Loss

VISUAL EXPLAINABILITY:
Grad-CAM Implementation: Visualize which image regions influenced prediction
Use Case: Help clinicians understand model reasoning
Medical Trust: Transparency increases adoption in clinical settings
